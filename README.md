## Overview
Connectify is a social networking platform built with the MERN stack, allowing users to create accounts, sign in, upload photos, connect with other users, and interact with posts through likes and comments. Additionally, it features a seamless switch between dark and light themes for an enhanced user experience.

## Features
- **User Authentication**: Secure signup and login processes.
- **Post Uploads**: Users can upload photos and pictures.
- **Social Interactions**: Users can like and comment on each other's posts.
- **User Connections**: Connect with friends and expand your network.
- **Theme Toggle**: Switch effortlessly between dark and light themes.
- **Seamless Navigation**: Integrated with React Router for smooth navigation.

## Tech Stack
- **Frontend**: React.js, MUI, Redux Toolkit, Formik, React Router
- **Backend**: Node.js, Express.js
- **Database**: MongoDB

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your_username/Connectify.git
   cd Connectify
   ```

2. **Install dependencies**
   ```bash
   # For backend
   cd backend
   npm install

   # For frontend
   cd ../client
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file in the `backend` directory with the following variables:
   ```plaintext
   DB_USERNAME=your_mongo_connection_username
   DB_PASSWORD=your_mongo_connection_password
   ```

4. **Run the application**
   ```bash
   # In the backend directory
   npm start

   # In the frontend directory
   npm start
   ```

## Usage
1. **Sign Up**: Create a new account using your email.
2. **Login**: Access your account with your email and password.
3. **Upload Posts**: Share photos and pictures.
4. **Interact with Posts**: Like and comment on posts from other users.
5. **Connect with Users**: Build your network by connecting with friends.
6. **Toggle Theme**: Switch between dark and light themes for a personalized experience.
